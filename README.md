# Future - 影视免费片源搜索工具

> 输入电影/动漫名称，全网搜索免费在线观看资源，找到后发送到邮箱，持续运行直到找到结果。

## 功能特点

- **多搜索引擎**：同时搜索 Google、Bing、百度
- **智能过滤**：自动识别免费观看相关结果，排除无关广告
- **优先排序**：正版平台（B站、腾讯视频、优酷、爱奇艺等）优先
- **页面验证**：自动验证搜索到的页面是否真的包含视频播放器
- **邮件通知**：每轮搜索实时发送结果到邮箱，找到片源后立即通知
- **持续运行**：默认每 2 分钟搜索一次，最多持续 72 分钟

## 安装依赖

```bash
pip install requests beautifulsoup4
```

## 使用方法

```bash
# 搜索电影（持续运行直到找到）
python future.py "流浪地球2"

# 搜索动漫
python future.py "进击的巨人"

# 自定义搜索参数
python future.py "灌篮高手" --max-retries 50 --interval 90

# 只搜索一轮
python future.py "三体" --once
```

### 参数说明

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `keyword` | 电影/动漫名称（必填） | - |
| `--max-retries` | 最大搜索轮数 | 36 (约72分钟) |
| `--interval` | 每轮间隔（秒） | 120 (2分钟) |
| `--once` | 只搜索一轮 | false |

## 工作流程

```
输入关键词 → 多引擎搜索 → 智能过滤打分 → 页面验证 → 邮件通知
     ↑                                                    |
     └────────────── 未找到则循环 ────────────────────────┘
```

## 邮件通知

- **搜索中**：每轮发送进度邮件，包含当前最佳搜索结果
- **找到片源**：发送确认邮件，附带可直接点击的观看链接
- **达到上限**：发送最终报告，包含所有已收集的结果

- 本工具仅搜索公开可用的免费影视资源
- 搜索结果仅供参考，请遵守当地法律法规
- 建议优先使用正版平台的免费内容
