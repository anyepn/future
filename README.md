# Future - 影视免费片源搜索工具

> 输入电影/动漫名称，全网搜索免费在线观看资源，找到后发送到邮箱。
>
> **三种使用方式**：命令行 / 邮件触发 / Web 网页

## 功能特点

- **多搜索引擎**：同时搜索 Google、Bing、百度
- **智能过滤**：自动识别免费观看相关结果，排除无关广告
- **优先排序**：正版平台（B站、腾讯视频、优酷、爱奇艺等）优先
- **页面验证**：自动验证搜索到的页面是否真的包含视频播放器
- **邮件通知**：每轮搜索实时发送结果到邮箱，找到片源后立即通知
- **三种触发方式**：命令行 / 发邮件 / 网页
- **持续运行**：默认每 2 分钟搜索一次，最多持续 72 分钟

## 安装依赖

```bash
pip install -r requirements.txt
```

---

## 方式一：命令行

```bash
# 搜索电影（持续运行直到找到）
python future.py "流浪地球2"

# 搜索动漫
python future.py "进击的巨人"

# 自定义搜索参数
python future.py "灌篮高手" --max-retries 50 --interval 90

# 只搜索一轮
python future.py "三体" --once
```

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `keyword` | 电影/动漫名称（必填） | - |
| `--max-retries` | 最大搜索轮数 | 36 (约72分钟) |
| `--interval` | 每轮间隔（秒） | 120 (2分钟) |
| `--once` | 只搜索一轮 | false |

## 方式二：邮件触发（推荐）

发送邮件即可自动搜索，结果回复到发件人邮箱。

### 启动邮件机器人

```bash
python email_bot.py
```

### 触发格式

| 邮件主题 | 效果 |
|----------|------|
| `搜: 流浪地球` | 搜索 1 轮，快速返回结果 |
| `搜: 进击的巨人 3轮` | 搜索 3 轮 |
| `搜: 灌篮高手 持续` | 持续搜索直到找到（最多 36 轮） |
| `搜索: 三体 5轮` | 同上（搜/搜索均可） |

### 使用场景

- **微信用户**：QQ 邮箱绑定微信后，微信里转发邮件即可触发
- **手机端**：在任意邮箱 App 中发送即可
- **自动化**：可实现定时搜索任务

## 方式三：Web 网页

### 启动 Web 服务

```bash
python web_server.py
```

然后访问 **http://localhost:5000**

### Web 功能

- 网页输入片名，实时显示搜索结果
- 结果可点击直接跳转观看
- 可选同时发送邮件通知
- 支持手机浏览器访问
